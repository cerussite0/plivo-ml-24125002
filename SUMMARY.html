🏆 Winning Architecture (Run 5)

  • BPE Tokenizer (vocab 1024 / 768 merges): Trained directly on corpus bytes, compressing Hindi Devanagari text by 5.0x and eliminating token inflation.
  • Rotary Positional Embeddings (RoPE): Parameter-free relative positioning, freeing 49.1K parameters for vocabulary expansion and doubling context block size to 512 tokens (~1,400 bytes).
  • SwiGLU MLP + RMSNorm: Gated non-linear activations with hidden dimension 512 and parameter-free root-mean-square normalization.
  • Tied Embedding Weights: Shares input embedding table weights with the output logit head.
  • AdamW Optimization: Cosine decay schedule (peak 6e-4, min 6e-5, warmup 100), weight decay 0.1, gradient accumulation = 8 (effective batch size 64).
